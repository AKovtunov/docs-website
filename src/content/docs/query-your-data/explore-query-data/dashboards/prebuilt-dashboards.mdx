Through the New Relic UI, you can access a collection of recommended pre-built dashboards, to help you instantly visualize your data for common use cases. These recommendations are personalized based on the technology stack of the account you have chosen.

Go to **[one.newrelic.com > All capabilities](https://one.newrelic.com/all-capabilities) > Dashboards** to open the dashboards page.

## How to install pre-built dashboards [#use]

To install a pre-built dashboard:

1. Select your account in the top right account picker.
2. From the dashboards page, search for the pre-built dashboard you want to customize.
3. Click the pre-built dashboard to select it. A new **dashboard setup** window should appear.
4. Follow the instructions. If you've already instrumented the data source, you can skip this step.
5.Click **View dashboard** to deploy the dashboard and start visualizing your data. Note that you can customize your new pre-built dashboard (see [Manage your dashboard](/docs/query-your-data/explore-query-data/dashboards/manage-your-dashboard/)). You can also change the name and settings, add new content and new widgets, and refine the charts

## How to view more templates [#more-templates]

Browse the entire dashboard catalog to find out-of-the-box dashboard templates for all available integrations. Click the **View All** link next to the **Recommended dashboards** title to see and search for all the available pre-built dashboards. You can use the <Icon name="fe-chevron-left"/> and <Icon name="fe-chevron-right"/> buttons to browse more pre-built dashboards.
