## Description

With this method, you can throw a demo run-time exception named `java.lang.RuntimeException` with a message. This crash will appear in the [**Crash Analysis** page](/docs/mobile-monitoring/mobile-monitoring-ui/crashes/crash-analysis-group-filter-your-crashes).
